# Telegram-калькулятор прибыли WB и Ozon

Бот считает ежедневную прибыль по Wildberries и Ozon:

- выручка;
- себестоимость;
- комиссия маркетплейса;
- эквайринг;
- логистика;
- хранение / приемка;
- реклама;
- налог;
- возвраты;
- прочие расходы;
- чистая прибыль и маржа.

## Запуск

1. Создайте бота через BotFather и получите токен.
2. Установите зависимости:

```bash
python3 -m pip install -r requirements.txt
```

3. Запустите бота:

```bash
BOT_TOKEN="ваш_токен" python3 bot.py
```

## Команды

```text
/start - помощь
/add - добавить день пошагово
/quick - пример быстрого ввода
/today - прибыль за сегодня
/summary - общая сводка
/month - сводка за текущий месяц
/history - последние записи
/delete ID - удалить запись
/export - выгрузить CSV
```

## Быстрый ввод

Формат:

```text
/quick площадка; товар; дата; шт; цена; себестоимость; комиссия%; эквайринг%; логистика; хранение; реклама; налог%; возвраты; прочее; заметка
```

Пример:

```text
/quick WB; триммер; сегодня; 10; 990; 420; 18; 1.5; 600; 120; 800; 6; 1; 0; акция
```

## Где хранятся данные

История сохраняется в:

```text
state/entries.json
```

Каждый пользователь видит только свои записи.
